Arvids Ubiquity Commands

  • up

    Moves up one dir, e.g.: from http://example.com/aaa/bbb/ccc.html to http://example.com/aaa/

  • base

    Goes to the root of the host, e.g.: from http://example.com/aaa/bbb/ccc.html to http://example.com/

  • coral

    Opens the coral cache mirror of this, e.g.: from http://example.com/aaa/bbb/ccc.html to http://example.com.nyud.net/aaa/bbb/ccc.html

  • src

    If the page contains an ubiquity command, "src" opens the source of that command in a new tab.
